# Dashboard Insights APIs

## Top Products
`GET /api/dashboard/top-products?limit=10`
Returns top selling products by value.

## Top Parties
`GET /api/dashboard/top-parties?limit=10`
Returns top customers and vendors.

## Customer Trend
`GET /api/dashboard/customer-trend`
Returns new vs existing customer counts.

## Login Activity
`GET /api/dashboard/login-activity`
Returns daily login frequency.

## Due Invoices
`GET /api/dashboard/due-invoices`
Returns a list of upcoming and overdue sale invoices.
